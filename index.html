Omni-Agent: Zero-Cost AI Orchestrator

⭐ Featured

A production-ready Multi-Agent CLI framework designed to execute security audits, auto-generate documentation, and perform investment analysis — all without expensive API token costs.

Built on a smart routing engine that dispatches tasks to local Ollama models (Llama 3, Mistral) for routine operations and escalates to cloud APIs (OpenAI / Anthropic) only for complex reasoning. This hybrid approach reduces inference costs by up to 90% while maintaining response quality. The system supports pluggable agent modules, making it extensible for any domain.

Security Research: Dify SSRF Vulnerability

⚠ CVE Research Medium-High Severity

In-depth analysis of a Server-Side Request Forgery (SSRF) vulnerability in Dify — a widely-used open-source AI platform with 50k+ GitHub stars.

Discovered that Dify's workflow executor could be manipulated via the base_url parameter to scan internal network topology, bypass IP-allowlist filters, and exfiltrate environment variables from the host. Wrote a full proof-of-concept exploit and responsibly disclosed the findings to the vendor with a detailed reproduction guide.

AI Meeting Scribe

An autonomous AI agent built from scratch that turns raw meeting recordings into structured, actionable documents in minutes.

The pipeline ingests audio files, transcribes speech using OpenAI Whisper, then feeds the transcript to an LLM agent that produces: a full transcript, an executive summary, a list of key decisions, and categorised action items with owners. Designed for remote teams where async communication is critical — eliminates the need to sit through recorded calls.

LangGraph Research Orchestrator

A personal research assistant powered by a LangGraph agentic workflow and a Streamlit interface, designed for iterative, hallucination-free data research.

Unlike a simple chatbot, this system uses a multi-step graph: a planning node decomposes the research question, parallel search nodes retrieve evidence from multiple sources, a verification node cross-checks facts, and a synthesis node produces a cited report. This mirrors the workflow of a junior data analyst — without the risk of AI confabulation. Configurable depth and source preferences via the Streamlit UI.
